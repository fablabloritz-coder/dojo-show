🔗 Configuration Start.gg

Connectez votre tournoi Start.gg pour importer automatiquement les jeux et joueurs.

📖 Comment trouver votre clé API ?

  1. Connectez-vous sur start.gg
  2. Allez dans Developer Settings : start.gg/admin/profile/developer
  3. Cliquez sur "Create new token"
  4. Donnez un nom au token (ex: "DOJO SHOW") et copiez la clé générée

📖 Comment trouver l'URL du tournoi ?

  1. Allez sur la page de votre tournoi sur start.gg
  2. Copiez l'URL depuis la barre d'adresse
  3. Format attendu : https://start.gg/tournament/nom-du-tournoi
